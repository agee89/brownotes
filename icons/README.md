# Icons untuk Brow Notes

Extension ini membutuhkan icon dalam berbagai ukuran. Anda perlu menambahkan file-file berikut:

## Icon Normal (Extension Aktif)
- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Disabled (Extension Nonaktif)
- `icon-disabled16.png` - 16x16 pixels
- `icon-disabled32.png` - 32x32 pixels
- `icon-disabled48.png` - 48x48 pixels
- `icon-disabled128.png` - 128x128 pixels

## Cara Membuat Icon

### Opsi 1: Menggunakan Tool Online
1. Buka https://www.canva.com atau https://www.figma.com
2. Buat design dengan ukuran 128x128 pixels
3. Gunakan emoji 📝 atau design sederhana dengan huruf "B"
4. Export dalam berbagai ukuran (16, 32, 48, 128)
5. Untuk icon disabled, gunakan warna abu-abu atau opacity 50%

### Opsi 2: Menggunakan Emoji sebagai Icon
1. Buka https://favicon.io/emoji-favicons/
2. Pilih emoji 📝 (memo) atau 📋 (clipboard)
3. Download dan rename sesuai ukuran yang dibutuhkan

### Opsi 3: Design Sederhana
Icon bisa berupa:
- Huruf "B" dengan background biru (#007bff) untuk aktif
- Huruf "B" dengan background abu-abu (#999) untuk disabled
- Atau icon notepad/memo sederhana

## Temporary Solution

Untuk testing sementara, Anda bisa:
1. Download icon placeholder dari https://via.placeholder.com/128/007bff/ffffff?text=B
2. Resize ke ukuran yang dibutuhkan
3. Atau gunakan icon dari extension lain sebagai placeholder

## Rekomendasi Design

**Icon Aktif:**
- Background: #007bff (biru)
- Foreground: #ffffff (putih)
- Symbol: Huruf "B" atau icon notepad

**Icon Disabled:**
- Background: #999999 (abu-abu)
- Foreground: #ffffff (putih)
- Symbol: Sama dengan icon aktif tapi dengan warna redup
